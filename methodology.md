# Investigation Methodology

## Phase 1 — Source Collection
Frontend source code was collected using browser-based inspection.

## Phase 2 — Keyword Enumeration
Keyword searches were performed recursively using command-line tools.

## Phase 3 — JavaScript Inspection
JavaScript files were extracted and manually reviewed for references and structural artifacts.

## Phase 4 — Infrastructure Validation
Subdomain enumeration and technology fingerprinting were conducted.

## Phase 5 — Historical Verification
Public archives were reviewed for historical frontend indicators.

## Phase 6 — Reporting
Findings were documented with timestamped evidence and structured reporting.
